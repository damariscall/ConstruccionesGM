# Construcciones Generales Modernas S.A. de C.V. - Sitio Web Oficial

Este repositorio contiene el código fuente del sitio web oficial de **Construcciones Generales Modernas S.A. de C.V.**, una plataforma diseñada para presentar los servicios de construcción, gestionar la identidad corporativa y facilitar la comunicación con clientes potenciales.

## 🚀 Demo en Vivo
Puedes ver el sitio desplegado aquí: [https://construcciones-gm.vercel.app/](https://construcciones-gm.vercel.app/)

## 🛠️ Tecnologías Utilizadas
El proyecto ha sido construido utilizando un stack moderno para garantizar rendimiento, escalabilidad y una experiencia de usuario fluida:

* **Frontend:** [Next.js](https://nextjs.org/) / [React](https://reactjs.org/) (desplegado en Vercel).
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/) para un diseño responsivo y moderno.
* **Backend:** Implementaciones con [FastAPI](https://fastapi.tiangolo.com/) para servicios de backend y automatización.
* **Infraestructura:** Despliegue continuo (CI/CD) mediante la integración de GitHub y Vercel.

## ✨ Características Principales
* **Diseño Profesional:** Interfaz limpia y optimizada para la navegación tanto en escritorio como en dispositivos móviles.
* **Catálogo de Servicios:** Sección detallada que describe las capacidades de la constructora.
* **Formularios de Contacto:** Integración para la captación de leads y consultas de proyectos.
* **Optimización SEO:** Estructura preparada para mejorar la visibilidad en motores de búsqueda.
* **Infraestructura Escalable:** Configuración preparada para integrar módulos de notificaciones masivas y gestión de bases de datos.

## 📦 Instalación y Configuración Local

Si deseas ejecutar este proyecto localmente, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/nombre-del-repositorio.git
    cd nombre-del-repositorio
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install
    # o si usas yarn
    yarn install
    ```

3.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    # o si usas yarn
    yarn dev
    ```

4.  **Abrir en el navegador:**
    Visita `http://localhost:3000` para ver los cambios en tiempo real.

## 🚀 Despliegue
El sitio está configurado para desplegarse automáticamente en **Vercel** al realizar un `push` a la rama `main`.

1. Conecta tu cuenta de Vercel con GitHub.
2. Selecciona este repositorio.
3. Vercel detectará automáticamente la configuración de Next.js y realizará el build.
